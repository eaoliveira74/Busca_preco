# Busca Preço

Landing page estática que destaca as melhores ofertas de TVs, aparelhos de som e celulares 5G coletadas nos marketplaces Mercado Livre e Amazon com links de afiliados.

## Recursos
- Interface responsiva com hero, abas e cartões organizados por fabricante e modelo.
- Botões de ação que fazem rolagem suave até as seções principais.
- Dados centralizados em `assets/js/app.js` para facilitar a atualização de preços e links.

## Como executar
Abra o arquivo `index.html` diretamente no navegador de sua preferência (basta dar duplo clique no arquivo ou usar o menu "Open With"). Não há dependências extras ou servidor necessário.

## Estrutura
- `index.html`: marcação principal e containers das abas.
- `assets/css/styles.css`: estilos globais, layout responsivo e animações sutis.
- `assets/js/app.js`: dados das ofertas e lógica das abas/rolagem.
